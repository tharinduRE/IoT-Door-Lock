# IoT-Door-Lock
IoT powered door lock using Arduino and ESP8266

This uses a Gateway built using a raspberry pi 3 running Node-Red

### Prerequisites

Raspberry Pi or Linux Server Running Node-RED

```
sudo npm install -g node-red
```
